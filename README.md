## ✨ Overview

ResumeForge is a modern resume builder that helps job seekers create professional and ATS-friendly resumes quickly and easily.

Whether you're a student, fresher, or experienced professional, ResumeForge provides a clean and intuitive interface to craft resumes that stand out.

✅ Modern Resume Templates

✅ ATS-Friendly Formatting

✅ Live Resume Preview

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/resumeforge.git
```

### Navigate to Project

```bash
cd resumeforge
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
